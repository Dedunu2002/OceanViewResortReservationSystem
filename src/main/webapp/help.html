Staff Quick Start Guide

For Receptionists

1. Registering Guests: Use the "New Booking" link to enter guest details. The system automatically calculates the bill based on room type.

2. Finding Records: Go to "View Reservations" and use the search bar to find guests by name if they forget their ID.

For Administrators

1. System Oversight: Access the Admin Dashboard to see daily revenue and total active bookings.

2. Maintenance: Use the "Update Rates" section (coming soon) to adjust seasonal pricing.

Common Issues

  • Login Failed: Ensure caps lock is off. Check with the Admin if your account is active.
  • Data Not Saving: Ensure all fields are filled before clicking "Submit."
